My name is Kåre Overgård. I’m a freelance IT consultant working primarily with Microsoft infrastructure, identity, and automation.

I’ve been working with Microsoft technologies for more than 30 years, starting with Windows 3.11 and NT, and later through the introduction of Active Directory, Entra ID (formerly Azure AD), and Microsoft 365. Over the years, I’ve supported and operated these platforms in a wide range of environments, helping organisations of different sizes with upgrades, migrations, and day‑to‑day challenges — both on‑premises and in the cloud.

This blog is a place to document practical, technical experiences from that work. The focus is on real‑world solutions and edge cases, not polished marketing material. Typical topics include:

- Active Directory and Entra ID  
- Conditional Access  
- Microsoft PKI  
- PowerShell automation  

Along the way, I’ve collected a fair number of lessons learned, workarounds, scripts, and the occasional “how did this even happen” story. Writing things down helps me, and if it also helps someone else avoid a problem or save some time, then the blog has served its purpose.

You can read more about my work at my [portfolio site (Danish)](https://overgard.dk).
